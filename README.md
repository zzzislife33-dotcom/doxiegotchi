# 🐾 Doxie-Gotchi

A retro-style Dachshund virtual pet game inspired by the classic Tamagotchi. Care for your digital "Doxie" in real-time right from your browser!

## 🕹️ Game Features
* **Dynamic Stats:** Manage Hunger, Happiness, and Cleanliness.
* **Visual Feedback:** Watch your pet's expression change and icons blink red when they need urgent care.
* **Smart Mechanics:** Status colors change (Green 🟢, Orange 🟠, Red 🔴) based on pet health.
* **Real-time Logic:** Stats decay every 30 seconds while playing, but the pet is "safe" when you return to the splash screen.
* **Retro UI:** Custom-built CSS device skin with a pixel-art aesthetic.

## 🛠️ Built With
* **HTML5** - Structure and Canvas
* **CSS3** - Custom device skin, animations, and masking
* **Vanilla JavaScript** - Game logic, state management, and Web Audio API

## 🚀 Quick Start
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Press any button to "Wake Up" your Doxie!

## 📜 How to Play
* **A Button:** Cycle through actions (Food, Play, Groom, Menu).
* **B Button:** Select/Confirm action.
* **C Button:** Open/Close Menu or Cancel.
